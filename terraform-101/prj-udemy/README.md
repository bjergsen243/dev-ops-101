# Terraform Course on Udemy

Link: https://www.udemy.com/course/learn-devops-infrastructure-automation-with-terraform/learn/lecture/5840342#overview

## Configure

Setup access key and secret key

```bash
aws configure
```

## Initialize

Init terraform

```bash
terraform init
```

## Plan

```bash
terraform plan
```

## Apply

```bash
terraform apply
```
